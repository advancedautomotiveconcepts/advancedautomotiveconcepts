## 2026-07-15 - Global Accessibility Scaffolding
**Learning:** Even simple static sites benefit significantly from consistent application of basic accessibility landmarks (skip links, `<main>` tags, and `aria-current` states). These provide the necessary "scaffolding" for assistive technologies to navigate predictably across multiple pages.
**Action:** Prioritize auditing for landmarks and navigation state (ARIA current) at the start of any UX mission, as these represent high-impact improvements that can often be implemented within a 50-line budget across the entire site.
